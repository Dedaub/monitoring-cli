## Proxies
- **V2: NO proxies.** Factory/Router/Pair immutable; pairs CREATE2-deployed (salt `keccak256(token0,token1)`) with the **Pancake-specific init code hash** (≠ Uniswap's, ≠ Sushi's; and the BSC hash ≠ the newer multichain-factory hash). Derive pairs off-chain per chain.
- MasterChef / CAKE: not proxies.

---

## Detection invariants & gotchas
1. **V2 topic0s are identical to Uniswap V2 AND Sushi V2** (all Uni V2 forks). Disambiguate Pancake vs Uniswap vs Sushi by the **factory/pair address**, never by topic0.
2. **Init code hash differs by chain**: BSC original `0x00fb7f63…` vs the newer `0x02a84c1b…` factory's hash on ETH/Base/Arbitrum. Use the right one per chain.
3. **MasterChef `Withdraw(address,uint256,uint256)` `0xf279e6a1…` collides** with Curve/Balancer/Sushi Withdraw topic0 — disambiguate by contract.
4. CAKE `symbol()` is `"Cake"` (not "CAKE").
