# QuickSwap V3 — Compressed Deployment Reference (Polygon PoS / Algebra V1)

**Status:** topic0/selectors computed locally via keccak-256 and **confirmed against live `eth_getLogs`** on the WPOL/USDC.e pool + NFPM (2026-06-05, publicnode); addresses verified via `eth_getCode`/`eth_call`. Pool init code hash verified by CREATE2 round-trip of two live pools.
**Scope:** QuickSwap **V3** — built on **Algebra V1** ("Algebra Finance", the pre-Integral dynamic-fee engine), **NOT a Uniswap V3 fork**. This was the flagship Algebra deployment. Other versions: [`v2.md`](v2.md) (Uniswap V2 fork), [`v4.md`](v4.md) (Algebra Integral, Base).
**Key facts:**
- **Deployed only on Polygon PoS (137)** among the 7 requested chains. QuickSwap's Algebra license was Polygon-exclusive; its other V3 deployments (Dogechain, Polygon zkEVM, X Layer) are outside this set. `eth_getCode` at the Algebra factory `0x411b…3A28` is `0x` on ETH/Base/BNB/AVAX/ARB/OP.
- **Algebra ≠ Uniswap V3.** Dynamic per-block fee (no fee tiers) → factory create has **no fee param** and the factory event is `Pool(token0,token1,pool)` (no `uint24 fee`). Fee lives in pool `globalState()` and emits a `Fee(uint16)` event. Pools are CREATE2-deployed by a separate **PoolDeployer** with **`abi.encode` salt** (not `encodePacked`).
- Pool `Swap`/`Mint`/`Burn`/`Collect`/`Initialize`/`Flash` topic0s **coincide with Uniswap V3** (same type tuples). The reliable Algebra discriminators are the factory `Pool` event (`0x91ccaa7a…`), the `Fee(uint16)` event (`0x598b9f04…`), the NFPM `IncreaseLiquidity` 6-arg form (`0x8a82de7f…`), and the NFT symbol **`ALGB-POS`** / name "Algebra Positions NFT-V1".

## Proxies

**Core is immutable** (Algebra V1, like Uniswap V3): AlgebraFactory, PoolDeployer, every Pool, SwapRouter, NFPM are non-upgradeable. Pools are CREATE2-deployed by PoolDeployer (init hash `0x6ec6c9c8…`, `abi.encode` salt) — not clones. The **mutable knobs are governance-controlled but not proxy upgrades**: `owner()` (`0xa08dac76…`) can set `farmingAddress`, `vaultAddress`, community fee, and each pool's dynamic-fee config; the pool's plugin/oracle is built-in (no external plugin in V1). Farming contracts (FarmingCenter/Eternal/Limit) are owner-swappable via the factory's `farmingAddress`, so the *active* farming entrypoint can change without redeploying the core.

## Detection invariants & gotchas

1. **Algebra V1 vs Uniswap V3 share Swap/Mint/Burn/Collect/Initialize/Flash topic0s.** Discriminate via: factory `Pool` event `0x91ccaa7a…` (no fee), the `Fee(uint16)` event `0x598b9f04…` (Algebra-only, frequent), or the NFPM 6-arg `IncreaseLiquidity` `0x8a82de7f…`. NFT `symbol()="ALGB-POS"`.
2. **Dynamic fee** — there is no fixed fee tier. Read the live fee from pool `globalState().fee` (3rd field, 1e-6 units); it changes block-to-block and emits `Fee(uint16)`. Do NOT assume 0.3%.
3. **No fee in pool identity.** A token pair maps to exactly ONE Algebra pool (`poolByPair(a,b)`), unlike Uniswap V3's per-fee pools. Off-chain derivation uses `abi.encode` salt + **PoolDeployer** address (not the factory).
4. **`Swap.amount0/amount1` are signed int256** (Algebra/UniV3 convention): negative = out of pool, positive = into pool. Mid-price field is `price` (sqrtPriceX96 format), `tick` is post-swap.
5. **Mint funds come via callback** (`algebraMintCallback`), not pre-transfer; `mint()` returns `liquidityActual` which can differ from requested for fee-on-transfer tokens.
6. **Built-in TWAP oracle** (timepoints in `globalState().timepointIndex`) — Algebra V1 has no separate oracle contract and emits no oracle event. (Integral moves this to a plugin — see `v4.md`.) Confirmed V1 (not Integral): the pool's `plugin()` call **reverts** (no plugin slot), `tickSpacing()=60`, and NFT name is "Algebra Positions NFT-V1".
7. **Polygon-only.** Don't query other chains for these addresses. The Base "V4" Algebra (`v4.md`) is a *different* engine version with different topics.
8. **Shared governance key** `0xa08dac76…` is the V3 factory `owner()`, V2 `feeToSetter`, and V4 factory `owner()` — one admin across all QuickSwap versions/chains.
