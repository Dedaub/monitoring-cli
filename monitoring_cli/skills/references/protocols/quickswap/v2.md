# QuickSwap V2 — Compressed Deployment Reference (Polygon PoS, Base)

**Status:** topic0/selectors computed locally via keccak-256 (pycryptodome) and confirmed against the deployed bytecode + `Uniswap/v2-core`/`v2-periphery` source; addresses verified live via `eth_getCode`/`eth_call` on publicnode RPCs (2026-06-05). Pair init code hash verified by CREATE2 round-trip of the live WPOL/USDC.e pair.
**Scope:** QuickSwap **V2** — the "classic" AMM, a **verbatim Uniswap V2 fork**. Concentrated-liquidity versions are separate: [`v3.md`](v3.md) (Algebra V1, Polygon), [`v4.md`](v4.md) (Algebra Integral, Base).
**Key facts:**
- QuickSwap V2 is a **byte-for-byte Uniswap V2 fork** (core unchanged): **topic0s and selectors are IDENTICAL to Uniswap V2** (see `uniswap/v2.md`), and the **pair init code hash is the SAME as Uniswap's** `0x96e8ac42…48845f` (QuickSwap deployed Uniswap's exact bytecode — unusual; most forks recompile to a different hash). LP token is still `name()="Uniswap V2"`, `symbol()="UNI-V2"`, 18 decimals, EIP-2612 permit.
- **Deployed on only 2 of the 7 requested chains: Polygon PoS (137) and Base (8453).** NOT on Ethereum, BNB, Avalanche, Arbitrum, or Optimism (all return `code=0` at the QuickSwap factory address). QuickSwap's Ethereum-L1 presence is a Liquidity-Hub aggregator + perps, NOT a V2 AMM.
- **Protocol fees are ON** on both chains: `feeTo()` is non-zero (`0x98081c…3348` on **both** Polygon and Base — same recipient).

## Proxies

**None.** Factory, Router02, and every Pair are immutable (verbatim Uniswap V2). Pairs are CREATE2-deployed (not minimal-proxy clones) — full bytecode each time. The only mutable knobs are `feeTo`/`feeToSetter`. dQUICK/QUICK token contracts are plain (non-proxy) ERC-20s.

## Detection invariants & gotchas

1. **Indistinguishable from Uniswap V2 by topic0/selector/init-hash.** The ONLY discriminator is the factory address per chain. Condition on `(chainId, 0x5757…/0xEC65…)`.
2. **`feeTo` is ON** on both chains (same recipient `0x9808…3348`) → protocol-fee LP mints occur (Pair `Transfer(0x0, feeTo, …)`). Exclude from organic-LP aggregates.
3. **LP token still reports `name="Uniswap V2"`, `symbol="UNI-V2"`** — do NOT use symbol to attribute liquidity to Uniswap; it's QuickSwap.
4. **Polygon WPOL ≠ rename event.** Same contract `0x0d50…1270`; only `symbol()` changed MATIC→POL. Router internally calls it `WETH()`.
5. **Not on ETH/BNB/AVAX/ARB/OP** — querying those chains for QuickSwap V2 yields nothing; don't assume cross-chain presence from the Polygon footprint.
6. **`Swap.sender` is the Router/aggregator, not the trader.** Same as Uniswap V2.
7. **Old vs new QUICK / dQUICK** coexist on-chain (both have live bytecode). Use the *new* `0xB5C0…eE17` (QUICK) and `0x958d…23A1` (dQUICK); the old pair is deprecated post-2022 1:1000 split.
